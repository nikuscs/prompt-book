# PromptBook Desktop (Tauri)

Desktop app package for PromptBook.

## Run

```bash
bun install
bun run tauri dev
```

## Validate

```bash
bun run check
```

## Build

```bash
bun run build
bun run tauri build
```

## Stack

- Tauri v2
- React 19 + Vite
- Tailwind CSS v4
- Rust + TypeScript strict checks
